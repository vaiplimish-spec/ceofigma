# ceofigma

Добро пожаловать в проект ceofigma!

## Описание

Этот проект содержит инструменты и ресурсы для работы с Figma.

## Начало работы

### Предварительные требования
- Node.js
- Git

### Установка

```bash
git clone https://github.com/vaiplimish-spec/ceofigma.git
cd ceofigma
npm install
```
